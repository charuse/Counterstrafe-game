# **Counterstrafe Game**
### Test your counterstrafing skill to get a high score!
### Written in Kotlin
Made for [Cloud9 x JetBrains Hackathon](https://cloud9.devpost.com/)

<br>

### How to build JAR file:
Open Gradle inside IntelliJ IDEA
- Tasks → shadow → run shadowJar
  - JAR located in `build/libs/`
